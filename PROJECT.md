# Diseno de un curso de ciberseguridad

## what it is?

we have access to 87 wargame exercises from a cybersecurity learning framework. the framework is a presented as a website that is hosted in a local network server dedicated for cybersecurity. 

The server is located at 192.168.22.28 and accessible with credencials `user`/`user`.

The exercises are clasiffied or grouped in categories such as: Crypto (13), Web Hacking (26), Forensics (20), System Hacking (12), Malware (3), ISMS (3), Reversing (10).

We are going to analyze each exercise and classify in groups for creating a 3-level (basic, intermediate, and advanced) of cybersecurity course.

After we choose the list for each difficulty level, we attempt to solve and document the course with all the necessary concepts and solution steps to complete the exercise.

## Final product

A website that presents the 3 level course based on these exercises.
