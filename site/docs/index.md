# Curso de Ciberseguridad — INICTEL

Este curso está construido a partir de ejercicios reales del servidor wargame de INICTEL. Cada ejercicio incluye la explicación del concepto, las herramientas necesarias y la solución paso a paso.

## Los tres niveles

| Nivel | Descripción |
|-------|-------------|
| 🟢 **Básico** | Un solo concepto. Sin herramientas especializadas. Ideal para comenzar. |
| 🟡 **Intermedio** | Combinación de 2-3 técnicas. Requiere haber completado algunos básicos. |
| 🔴 **Avanzado** | Cadena de técnicas. Requiere dominio de los niveles anteriores. |

## Cómo trabajar un ejercicio

1. **Lee la teoría** — entiende el concepto antes de intentar resolver
2. **Conéctate al servidor** — ver [Conexión al servidor](./conexion)
3. **Intenta resolverlo tú mismo** — usa las pistas si te atascas
4. **Revela la solución** — compara tu enfoque con el writeup

:::tip[Consejo]
No copies la solución directamente. El valor está en entender *por qué* cada paso funciona, no en obtener la flag.
:::
