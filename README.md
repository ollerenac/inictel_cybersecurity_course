# Curso de Ciberseguridad — INICTEL

Curso de ciberseguridad estructurado en 3 niveles (básico, intermedio, avanzado), basado en ejercicios reales del framework wargame de INICTEL. Cada ejercicio incluye teoría del concepto, herramientas utilizadas y solución paso a paso, todo en español.

## Sitio web

**https://ollerenac.github.io/inictel_cybersecurity_course/**

## Categorías

- Criptografía
- Web Hacking
- Forensics
- System Hacking
- Malware
- ISMS
- Reversing

## Desarrollo local

```bash
cd site
npm install
npm start
```

El sitio queda disponible en `http://localhost:3000`.
